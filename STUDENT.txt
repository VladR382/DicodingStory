## Cara Menjalankan Aplikasi
1. Buka Terminal Jalankan `npm install` untuk menginstal dependensi
2. Jalankan `npm run dev` untuk menjalankan aplikasi dalam mode development
3. Akses aplikasi di `http://localhost:5173/`
